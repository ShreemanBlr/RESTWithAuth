Hello, Welcome to Perpule

This is a REST Endpoint application with token-based authentication enabled

To add a customer record use the below link with Basic Auth option from a REST client (Postman) to trigger a POST request

https://perpule-cust-ops.appspot.com/rest/customer/add


To query a customer record use the below link with Basic Auth option from a REST client (Postman) to trigger a GET request

https://perpule-cust-ops.appspot.com/rest/get/customer?name="QueryName"&phone="QueryPhone"
